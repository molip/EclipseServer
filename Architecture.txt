MongooseServer: ports, connections. 
Forwards HTML and WS events to Server (port). 
Forwards messages from Server (port) to client (connection).

Server: ports, playerIDs, controller. 
Maintains list of online players (port <-> playerID) and notifies controller on change. 
Prevents duplicate player IDs. 
Forwards messages (port) to controller (playerID).
Forwards messages from controller (playerID) to MongooseServer (port).
Serves game HTML. 

Controller: playerIDs, games, commands.
Sends commands to games (playerID). 
Sends messages to server (playerID). 

Model:
Maintains game list.
